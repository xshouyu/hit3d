# HIT3D 使用说明

HIT3D 是一个基于伪谱方法的三维均匀各向同性不可压缩湍流直接数值模拟（DNS）程序，使用 Fortran 90 编写，通过 MPI 并行，FFT 计算由 FFTW3 库完成。

## 1. 依赖环境

| 依赖 | 说明 |
|------|------|
| gfortran | Fortran 90 编译器 |
| mpif90 / OpenMPI | MPI 并行通信库 |
| FFTW3 | 快速傅里叶变换库 |

Ubuntu/Debian 安装命令：

```bash
sudo apt install gfortran openmpi-bin libopenmpi-dev libfftw3-dev
```

---

## 2. 编译

```bash
cd ~/hit3d/source/
make
```

生成可执行文件 `hit3d.x`。

清理编译产物：

```bash
make clean
```

**当前 Makefile 关键编译选项说明：**

| 选项 | 作用 |
|------|------|
| `-O3` | 开启高级编译优化 |
| `-fallow-argument-mismatch` | 允许旧式 MPI 调用中的参数类型宽松匹配（gfortran 10+ 必需） |
| `-I/usr/include` | FFTW3 头文件路径 |
| `-L/usr/lib/x86_64-linux-gnu -lfftw3` | 链接 FFTW3 库 |

---

## 3. 运行

### 准备运行目录

```bash
mkdir -p ~/hit3d/run
cp ~/hit3d/source/hit3d.x ~/hit3d/run/
cp ~/hit3d/examples/00_example.in ~/hit3d/run/run.in
cd ~/hit3d/run/
```

### 启动模拟

程序从**标准输入**读取参数文件，使用 MPI 多进程启动：

```bash
mpirun -np 4 ./hit3d.x < run.in
```

`-np 4` 为 MPI 进程数，建议设置为物理 CPU 核数。

### 从断点重启

将输入文件中的 `ITMIN` 设置为已保存的重启文件时间步编号（非 0），程序将从该时刻继续计算。

## 4. 输入文件详解

输入文件为纯文本，程序按行顺序读取数值，`#` 开头为注释，`---` 分隔线会被忽略。

以下以 `examples/00_example.in` 为参考逐项说明。

---

### 4.1 网格分辨率

```
64 64 64        NX, NY, NZ
```

三个方向的网格点数。计算域固定为 **2π × 2π × 2π** 的周期性立方体。

- 网格越大，分辨率越高，计算量按 **N³ log N** 增长
- 常用分辨率：32³、64³、128³、256³、512³

---

### 4.2 时间步控制

```
000000          ITMIN    起始时间步（0=全新模拟，>0=从重启文件恢复）
200000          ITMAX    最大时间步数（达到此步数后停止）
10              IPRNT1   每隔多少步计算并输出统计量
2000            IPRNT2   每隔多少步写一次重启文件
500             IWRITE4  每隔多少步写一次 real*4 二进制后处理文件
```

---

### 4.3 模拟时间控制

```
10.0            TMAX        模拟物理时间上限（不是墙钟时间）
1.0  3          TRESCALE, NRESCALE
0.0             TSCALAR     被动标量开始运动的时刻
```

- **TMAX**：当模拟时间（积分时间）达到此值时停止，与时间步数 ITMAX 取先到者。
- **TRESCALE / NRESCALE**：在物理时间 `TRESCALE` 时对速度场进行能量归一化（rescale），共执行 `NRESCALE` 次。常用于衰减湍流：先让流场建立相关性，再归一化以获得更高动能初始态。
- **TSCALAR**：被动标量从此时刻起随流场运动（之前冻结）。

---

### 4.4 流动类型

```
1               ITYPV    0=衰减湍流  1=强迫湍流
```

- **0（衰减湍流）**：不施加外力，湍流能量随时间自然衰减。
- **1（强迫湍流）**：在低波数处持续注入能量，维持湍流统计稳态。

---

### 4.5 物理参数

```
250.0           RE      雷诺数（= 1/ν，ν 为运动粘度）
-0.0005         DT      时间步长
```

- **RE**：雷诺数越大，湍流越强，需要越高分辨率才能分辨所有尺度。
- **DT**：
  - 负值：使用固定时间步长 |DT|
  - 正值：自适应时间步长，由稳定性条件（CFL）动态确定，DT 作为上限

---

### 4.6 初始速度谱

```
0               ISPCV1   初始能量谱类型（见 init_velocity.f90）
4               mv1      低波数段红外指数（谱斜率）
2.0             wm0v1    峰值波数（能量集中的尺度）
```

初始速度场在傅里叶空间按指定能量谱随机生成。`mv1` 控制大尺度（低波数）的能量分布斜率，`wm0v1` 决定能量最集中的波数。

---

### 4.7 强迫参数（仅 ITYPV=1 时有效）

```
1               force_type   强迫类型（目前仅支持 1）
3               KFMAX        强迫波数范围：k = 1 到 KFMAX
1               FAMP         能量注入幅度（通常设为 0.5~1.0）
```

- **force_type=1**：来自 Michael's PRL 论文（PRL #79(18) p.3411）的强迫方案，在低波数壳层中注入固定能量。
- **KFMAX**：只对波数 k ≤ KFMAX 的模态施加强迫。
- **FAMP**：控制能量输入速率，影响湍流强度的稳态值。

---

### 4.8 去混叠方案

```
1               dealias    0=2/3截断规则  1=相移+截断
```

伪谱方法中非线性项会产生混叠误差，需要去除：

| 选项 | 方法 | 速度 | 有效模态数 |
|------|------|------|-----------|
| `0` | 2/3 截断规则 | 快 | 较少（约 N/3 个波数） |
| `1` | 相移 + 截断 | 慢约 60% | 明显更多，DNS 最优 |

对于高精度 DNS，推荐使用 `1`。

---

### 4.9 随机数种子

```
0               det_rand    （当前版本此参数无效，种子始终从文件读取）
67748937.       RN1
93712139.       RN2
10765397.       RN3
```

固定种子可确保不同分辨率下产生相似的大尺度流场结构，便于对比研究。

---

### 4.10 拉格朗日粒子追踪

```
0               np                   粒子数量（0=不使用粒子）
1               particle_tracking    插值方式：0=三线性  1=4点三次插值
0.1             time_p               粒子释放时刻（物理时间）
0.3             particle_filter_size 速度场滤波尺度（0.0=不滤波）
```

粒子随流场被动输运，用于收集拉格朗日统计量（如粒子加速度、扩散等）。

- **particle_tracking=1**（三次插值）精度更高，计算量稍大。
- **particle_filter_size**：粒子可以被局部平均速度（而非完全分辨速度）驱动，用于 LES 粒子研究。

---

### 4.11 LES 模型

```
0               les_model    0=纯 DNS（无模型）
```

当前版本 LES 模型尚未调试完成，**建议保持为 0**，以 DNS 模式运行。

---

### 4.12 被动标量

```
1               nums         被动标量数量（0=不使用）
```

若 `nums > 0`，在下方每行添加一个标量的参数：

```
类型   Schmidt数   红外指数   峰值波数
3      1.0         4.0        2.
```

**标量类型说明：**

| 类型值 | 初始条件 |
|--------|---------|
| `0` | 均值梯度强迫标量 |
| `1` | 指数谱（傅里叶空间） |
| `2` | von Karman 谱（傅里叶空间） |
| `3` | 双 delta PDF（傅里叶空间） |
| `11` | 单层平板（实空间） |
| `12` | 双层平板 |
| `13` | 一维正弦波 |

**Schmidt 数**（Sc = ν/D）：粘度与标量扩散系数之比。Sc=1 表示动量与标量扩散速率相同。

## 5. 输出文件说明

| 文件类型 | 触发条件 | 用途 |
|----------|---------|------|
| 统计量文件 | 每 `IPRNT1` 步 | 能量、耗散率、雷诺数等时间序列 |
| 重启文件（`.64.NNNNNN`） | 每 `IPRNT2` 步 | 双精度完整流场，用于断点续算 |
| 后处理文件（real*4） | 每 `IWRITE4` 步 | 单精度流场快照，用于可视化和分析 |

重启文件命名格式：`运行名.64.时间步数`，例如 `hit3d.64.002000`。
